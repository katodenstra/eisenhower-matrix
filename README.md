# Eisenhower Matrix (Angular)

A friendly (rounded) Eisenhower priority matrix app:

- 4 quadrants (Do Now / Do Later / Delegate / Eliminate)
- Task creation + editing
- Expand/collapse quadrant
- Expand/collapse task (animated)
- Drag & drop between quadrants
- Drag & drop sorting inside quadrant (animated)
- Uncategorized inbox lane
- Local persistence (localStorage)
- "Add to calendar" exports an .ics event

## Run

```bash
npm i
npm start
```
